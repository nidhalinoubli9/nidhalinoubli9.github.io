Cypress E2E Tests

Project Overview

This project demonstrates end-to-end (E2E) testing for web applications using Cypress. It includes automated test scripts that cover critical workflows from login to checkout and user interactions.

Key Features

  • Automated tests for main user flows
  • Assertions for UI elements and functionality
  • Reusable commands for consistent testing
  • Integration with CI/CD pipelines for automated runs
