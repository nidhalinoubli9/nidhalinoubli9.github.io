Selenium Login Tests

Project Overview

This project contains automated login tests for web applications using Selenium WebDriver. It includes test scripts for various login scenarios, error validations, and cross-browser testing.

Key Features

  • Automated login with valid and invalid credentials
  • Error message verification
  • Cross-browser testing (Chrome, Firefox)
  • Reusable functions and Page Object Model implementation
