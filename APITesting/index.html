API Testing with Postman

Project Overview

This project demonstrates REST API testing using Postman. It includes collections and scripts for automated API validation, functional testing, and integration tests for web applications.

Key Features

  • Automated API requests and validations
  • Environment variables for flexible testing
  • Pre-request and test scripts for automation
  • Integration with CI/CD pipelines for continuous testing
